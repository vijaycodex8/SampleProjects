## 📂 Projects Included

### 1. AI Interview Preparation Bot
Conversational AI bot powered by NLP and LLMs that generates role-specific interview questions, evaluates candidate answers in real-time, gives smart feedback, and tracks session performance. FastAPI backend with prompt-engineered OpenAI integration and Bootstrap UI.

### 2. Blockchain-Based Secure Digital Identity System
Decentralised digital identity platform using Python and blockchain — users own their credentials, verifiers query on-chain proofs with no central authority. JWT-secured FastAPI REST layer with cryptographic signing and Decentralised ID (DID) standards compliance.

### 3. AI-Based Resume Screening for Government Hiring
NLP-powered recruitment pipeline that parses resumes, ranks candidates against government job criteria using ML classifiers, flags eligibility automatically, and generates shortlist reports — replacing manual screening for high-volume public sector recruitment.

### 4. ERP Management System — Django Enterprise
Comprehensive university ERP built with Django — covers student management, faculty records, attendance tracking, exam management, fee collection, and admin dashboards. Role-based access for students, staff, and administrators with REST API integrations and Sqlite backend.

### 5. GarmentInsight AI — Supply Chain & CRM Analytics
Unified fullstack platform for garment businesses: real-time supply chain tracking, AI-driven demand forecasting, CRM with customer segmentation, and an interactive analytics dashboard. Django + DRF backend with Bootstrap frontend and Pandas-powered ML insights.

### 6. Generative AI for Cyber Security — Threat Analysis Platform
Production platform analysing how ChatGPT, DALL-E and generative AI models enhance cybersecurity — threat intelligence summarisation, attack pattern generation for red-teaming, and automated vulnerability report drafting via FastAPI + LLM pipeline.
